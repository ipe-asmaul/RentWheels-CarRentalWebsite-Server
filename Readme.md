## Api End Points:
* https://rent-wheel-server.vercel.app/cars - For fetching all Cars
* https://rent-wheel-server.vercel.app/featured - Featured Limit to 6 at Newest added
* https://rent-wheel-server.vercel.app/listings?id&email= - Added Cars list but need authorization
* https://rent-wheel-server.vercel.app/mybookedcars?id&email= - Booked Individual Cars list but need authorization
* https://rent-wheel-server.vercel.app/car/id= - Specific car details
